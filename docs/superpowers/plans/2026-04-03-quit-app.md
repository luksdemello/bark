# Quit App Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Adicionar opção de encerrar o app Bark via menu de contexto no tray (clique direito) e botão no header da janela.

**Architecture:** Um comando Tauri `quit_app` em Rust centraliza o encerramento (`app.exit(0)`). O menu nativo do tray chama `app.exit(0)` diretamente no handler Rust. O botão no header chama `invoke("quit_app")` via frontend.

**Tech Stack:** Tauri v2 (Rust) — `tauri::menu::{Menu, MenuItem}`, `tauri::tray::TrayIconBuilder`; React/TypeScript — `@tauri-apps/api/core` `invoke`
